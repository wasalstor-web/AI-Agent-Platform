📋 كيفية الاستخدام | How to Use

  1. افتح http://localhost:3000 في المتصفح
  2. أنشئ حساباً جديداً (أول مستخدم = مدير)
  3. اختر أحد النماذج الستة
  4. ابدأ المحادثة مع الذكاء الصناعي!

🛠️ أوامر إدارة النظام | System Management Commands

  1. View logs: docker logs -f open-webui
  2. Restart: docker restart open-webui
  3. Stop: docker stop open-webui
  4. Start: docker start open-webui
  5. Check status: docker ps | grep open-webui
